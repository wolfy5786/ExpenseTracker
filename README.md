## 🚀 Features

- 🔐 **User Authentication** using **JWT**
- ➕ Add, 🗑️ Delete, and ✏️ Update expenses
- 🔎 Filter expenses by:
  - 📅 **Date Range**
  - 🏷️ **Category**
  - 💸 **Highest & Lowest Spend**
- 📊 Sort expenses by amount or date

---

## 🛠️ Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Docker

### Frontend
- React
- Vite
- Axios
- Tailwind CSS
